## 本日の学習内容

### Chapter 5-5: for文の繰り返し処理

- **for文の基本構文**

```javascript
for (let i = 1; i <= 10; i++){
    console.log('こんにちは！');
}
```

- 初期化、条件式、増減式の3つの要素で構成
- 同じ処理を複数回繰り返すための構文


### Chapter 5-5: インクリメント・デクリメント

- **インクリメント**: 値を1増やす

```javascript
let j = 0;
j++;  // ポストインクリメント（表示してから+1）
++j;  // プリインクリメント（+1してから表示）
```

- **デクリメント**: 値を1減らす

```javascript
j--;  // ポストデクリメント
--j;  // プリデクリメント
```

- ポストとプリの違い：値を返すタイミングが異なる


### Chapter 5-6: 配列とfor文で画像一覧表示

- **配列のインデックス**を使ってfor文で順に処理

```javascript
const lists = ['strawberry.jpg', 'lime.jpg', 'mango.jpg'];

for (let i = 0; i < lists.length; i++){
    const content = `<div><img src="images/${lists[i]}" alt=""></div>`;
    menu.insertAdjacentHTML('beforeend', content);
}
```

- `lists.length`で配列の要素数を取得
- `lists[i]`でi番目の要素にアクセス


### Chapter 5-7: let と const の違い

- **let**: 再代入が可能

```javascript
let letName = '太郎';
letName = '花子';  // ✅ OK
```

- **const**: 再代入が不可能

```javascript
const constName = '太郎';
constName = '花子';  // ❌ エラー
```

- **const配列の特徴**: 配列自体は変更できないが、要素は変更可能

```javascript
const sampleArray = ['りんご', 'ばなな', 'ぶどう'];
sampleArray[0] = 'いちご';  // ✅ OK（要素の変更）
sampleArray = ['すいか'];   // ❌ エラー（配列の再代入）
```


### Chapter 5-8～5-9: オブジェクトとキーバリューペア

- **オブジェクト**: 複数のデータを名前（キー）付きで管理

```javascript
const strawberry = {
    name: 'イチゴ',
    img: 'strawberry.jpg',
    price: 450
};
```

- **キーバリューペア**: キーと値の組み合わせ
- **値の取得方法**:
    - ドット記法: `strawberry.name`
    - ブラケット記法: `strawberry['name']`


### Chapter 5-10～5-11: 配列とオブジェクトの組み合わせ

- **配列にオブジェクトを格納**して複雑なデータを管理

```javascript
const lists = [
    {name: 'イチゴ', img: 'strawberry.jpg', price: 450},
    {name: 'ライム', img: 'lime.jpg', price: 400},
];

for (let i = 0; i < lists.length; i++){
    const content = `<div>
        <img src="images/${lists[i].img}" alt="">
        <h2>${lists[i].name}</h2>
        <p>${lists[i].price}円</p>
    </div>`;
    menu.insertAdjacentHTML('beforeend', content);
}
```

- `lists[i].img`で配列のi番目のオブジェクトのimgキーにアクセス


### Chapter 5-12: 分割代入でコードをスッキリ

- **分割代入**: オブジェクトのプロパティを一度に複数の変数に代入

```javascript
// 通常の書き方（3行）
const name = lists[i].name;
const img = lists[i].img;
const price = lists[i].price;

// 分割代入（1行）
const {name, img, price} = lists[i];
```

- キー名と変数名が一致する必要がある
- モダンなJavaScriptの書き方（ES6以降）


### Chapter 6-1～6-5: アニメーションの基礎

- **Web Animations API**: JavaScriptでアニメーションを実装

```javascript
const heading = document.querySelector('#heading');

const keyframes = {
    opacity: [0, 1],
};

heading.animate(keyframes, 2000);
```

- **基本構文**: `動かす要素.animate(動かす内容, 再生時間)`
- **キーフレーム**: 開始と終了の状態を指定すると、中間が自動補完される

## スコープの理解

- **スコープ**: 変数が有効な範囲
- **let/const**: ブロックスコープ（`{}`の中でのみ有効）

```javascript
for (let i = 0; i < 10; i++){
    const {name, img, price} = lists[i];  // スコープ内
    console.log(name);  // ✅ OK
}
console.log(name);  // ❌ エラー（スコープ外）
console.log(lists[0].name);  // ✅ OK（元のデータ）
```

## 疑問点・重要ポイント

- for文の無限ループに注意（条件式が常にtrueにならないように）
- constは再代入不可だが、配列の要素は変更可能
- 分割代入のキー名と変数名は一致させる必要がある
- ブラケット記法は変数をキーとして使う場合に必要
- スコープ外の変数は参照できない（元のデータは参照可能）
