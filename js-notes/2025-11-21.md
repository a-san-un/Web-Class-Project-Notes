### Chapter1: JavaScriptの基本構文

JavaScriptの基本的な書き方と、オブジェクト・メソッド・パラメーターの概念を学習しました。

#### オブジェクト・メソッド・パラメーター

- **オブジェクト(Object)**: 「物」という意味で、動作させる対象を指示する
- **メソッド(Method)**: 「方法」という意味で、「動作・命令」を表現する
- **パラメーター(Parameter)**: 「調整値」という意味で、メソッドに与える補足情報


### Chapter2: コンソール出力

開発者ツールのコンソールへの出力方法を学習しました。

***

## コード例

### 基本的なアラート表示

```javascript
// windowオブジェクトのalertメソッド
window.alert('はじめてのJavaScript!');

// windowは省略可能
alert('はじめてのJavaScript!');
```


### コンソールへの出力

```javascript
// コンソールにメッセージや値を出力
console.log('コンソールにメッセージを表示');

// オブジェクトのプロパティ一覧をツリー表示
console.dir(window);
```

## ポイント

- `console.log()`: コンソールにメッセージや値を出力するメソッド
- `console.dir()`: 渡したオブジェクトのプロパティ一覧をツリー表示するメソッド
- `window`オブジェクトはブラウザの最上位オブジェクトで、省略して記述できる
